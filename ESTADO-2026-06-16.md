## Como funciona todo (coordinacion) -- DOS VERTIENTES

Entrada: humano 21 landmarks / robot qpos -> FK -> **STAGE-2 Dong (compartido)** ->
quats por joint + chain + tips (wrist-local). De ahi se bifurca:
- **VERTIENTE A (latente/red)**: humano pose 6D -> E_h ; robot qpos -> E_r -> z compartido
  -> decoder -> qpos. (Zhou: 6D continuo para la red. Robot entra qpos crudo, NO rotacion.)
- **VERTIENTE B (metrica S_k)**: de los MISMOS quats -> tips/pinch (Xin) + stage-3 ->
  vector UDHM angulos -> termino udhm. (antes: quats -> D_R bundleado)

**El pegamento = el triplet.** S_k NO toca el 6D ni el latente. S_k (de la geometria)
RANKEA: para un anchor, la pose mas similar = positivo. El triplet jala z_pos al anchor,
empuja z_neg. O sea: **la similitud DEFINE donde cae cada embedding 6D en el latente.**
6D = entrada que se embebe; UDHM-angles = maestro que dice como arreglar los z.
Cambiar D_R->UDHM = hacer a S_k mas listo sobre abduccion -> el latente reorganiza para
separar poses que difieren en spread -> el decoder aprende la abduccion correcta.
